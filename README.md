## 🛠️ 客户端进阶：如何配置 Fake-IP 模式防止 DNS 泄露？

在接入 `KC云加速` 的高速节点时，建议在客户端配置文件中启用 `Fake-IP` 模式。这不仅能缩短域名解析的握手时间，还能有效防止本地真实 DNS 泄露导致的部分网站无法访问：

```yaml
# 推荐置于配置文件 dns 模块内
dns:
  enable: true
  enhanced-mode: fake-ip
  fake-ip-range: 198.18.0.1/16
  default-nameserver:
    - 119.29.29.29
    - 223.5.5.5
