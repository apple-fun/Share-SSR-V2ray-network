## 🛠️ 客户端常见报错与配置优化 (进阶必看)

### 1. 为什么导入订阅后无法联网？
* **检查系统时间：** 自建节点或 V2ray 协议对系统时间要求极高，本地时间与标准时间相差超过 1 分钟会导致握手失败。
* **代理端口冲突：** 确保客户端设置的 `Mixed Port` (通常为 7890) 没有被本地其他软件（如第三方杀毒或某些开发环境）占用。

### 2. 核心分流规则集 (Rule-Based)
