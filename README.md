## ⚙️ 第二部分：软路由网关防 DNS 污染与全局高并发分流配置

为了避免软路由接管全家或全公司流量后，国内本地流量（如微信、淘宝、国内视频网站）误走代理导致无谓消耗，请务必在 OpenClash 或相关插件的自定义规则中启用以下 **Geosite / GeoIP 核心混淆分流策略**：

```yaml
# 推荐置于网关客户端配置文件 rules 模块顶部
dns:
  enable: true
  enhanced-mode: fake-ip
  fake-ip-range: 198.18.0.1/16
  default-nameserver:
    - 119.29.29.29
    - 223.5.5.5
rules:
  - DOMAIN-SET,apps-ai-services,PROXY      # AI 生产力工具走专线
  - GEOSITE,category-ads-all,REJECT        # 自动拦截全网隐私追踪与弹窗广告
  - GEOSITE,cn,DIRECT                       # 国内域名直连
  - GEOIP,CN,DIRECT                         # 国内 IP 直连
  - MATCH,PROXY                             # 剩余流量默认走专线加速

```
